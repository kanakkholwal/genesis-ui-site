Accordion

Accordion is a vertically collapsing element to show and hide content via class changes.

Basic example

Click the accordions below to expand/collapse the accordion content.

Accordion Header 1
This is the first item's accordion body. It is hidden by default, until the collapse plugin adds the appropriate classes that we use to style each element. These classes control the overall appearance, as well as the showing and hiding via CSS transitions. You can modify any of this with custom CSS or overriding our default variables. It's also worth noting that just about any HTML can go within the .G_Accordion_Body-content, though the transition does limit overflow.

Open Multiple Items

just add accordion-multiple="true" attribute to parent element with .G_Accordion class to enable multiple accordion to open at a time .
