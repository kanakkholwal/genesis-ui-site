Badges

Responsive Badges add extra information like count or label to any content. Use counters, icons, or labels.

Basic example

Use .Badge class to provide your visitors with highlighted, new or unread items by adding it to links, text navs, and more.

Example heading New

Codepen 
												
                                    <h2>Example heading <span class="Badge">New</span></h2>

Sizes

Badges scale to match the size of the immediate parent element by using relative font sizing and em units.

Button

Badges can be used as part of links or buttons to provide a counter.

Codepen 
												
                                    <button type="button" class="btn btn-primary">
                                    Notifications<span class="Badge Badge_danger">8</span>
                                </button>

Positioned

Add .Badge_positioned class to parent element of the badge and adjust badge position using classes like : .TopLeft .TopCenter .TopRight .BottomLeft .BottomCenter .BottomRight .LeftCenter .RightCenter .

Animated

Add .Badge_* class to .Badge Element to animate with these animations : .animate_bounce .animate_flash .animate_pulse .animate_rubberBand .animate_shakeX .animate_shakeY .animate_headShake .animate_ripple .
